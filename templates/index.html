Climate Information

This website provides several different links that can be used to gather different information about the weather in Hawaii.The dataset runs ends in 2017 so the date is not currently up to date.

Percipitation

URL - /api/v1.0/precipitation

This link will return json with the percipitation by day for the last year

Station Information

URL - /api/v1.0/stations

This link will return json with all the station names and id's

Temperature Observations

URL - /api/v1.0/tobs

This link returns something with weather

T Stats with Start Date

URL - /api/v1.0/<start>

This link returns the T stats for all days greater then the start date . This url requires a start date. The date is formated as such YYYY-MM-DD.

T Stats with Start Date and End Date

URL - /api/v1.0/<start>/<end>

This link returns the T stats for all days between the start and end date inclusivly . This url requires a start date and an enddate. The date is formated as such YYYY-MM-DD.
