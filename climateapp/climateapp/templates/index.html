Climate Information

This website provides several different links that can be used to gather different information about the weather in Hawaii. The dataset runs till August 23rd, 2017, so the data set is not currently up to date. You can preview the data by clinking on the link in the side bar.

Percipitation

Endpoint - /api/v1.0/precipitation

This endpoint will return a json with the percipitation by day for the last year. No parameters are required for this endpoint.

Station Information

Endpoint - /api/v1.0/stations

This endpoint will return a json with all the station names and id's. No Paramaters are required for this endpoint.

Temperature Observations

Endpoint - /api/v1.0/tobs

This endpoint returns the T stats for the top ranked weather station in Hawaii. Top ranked is defined by having the most observations within the data set. No parameters are required for this endpoint.

Min Max and Average with Start Date

Endpoint - /api/v1.0/<start>

This endpoint returns the min, max and average for all days greater then the start date . This url requires a start date. The date is formated as such, YYYY-MM-DD. The link in the side bar currently returns null if you click it. But the endpoint will return results if you pass a start date to the endpoint itself.

Min Max and Average with Start Date and End Date

Endpoint - /api/v1.0/<start>/<end>

This endpoint returns the min, max and average for all days between the start and end date inclusivly . This endpoint requires a start date and an end date. The date is formated as such, YYYY-MM-DD. The link in the side bar currently returns null if you click it. But the endpoint will return results if you pass a start date and an end date to the endpoint itself.
